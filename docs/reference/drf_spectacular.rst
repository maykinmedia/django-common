===============
drf-spectacular
===============

`drf-spectacular <http://drf-spectacular.readthedocs.io/>`_ is an excellent library
for OpenAPI 3.x schema generation from your django-rest-framework endpoints.

The utilities documented here provide additional tooling or fixes/workarounds for those
generated schemas.
